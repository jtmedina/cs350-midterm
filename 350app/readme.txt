Purpose: The purpose of this project is to demonstate the use of a start menu in a python program. The start menu is always
running and it is the first function to be used. It then calls the other functions depending on the user's input, and the user can quit
the current game at any time. The included games in this demonstration are: Coin Flip, Dice Roller, and Guess the Number.

Libraries used: sys, random

This application is most useful for entertainment purposes.
